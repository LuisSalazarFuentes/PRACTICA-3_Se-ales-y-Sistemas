1.11-1 Anonymous Functions

Una función anónima proporciona una representación simbólica de una función definida en términos de operadores, un ejemplo de ello es la siguiente funcion
f = @(t) exp(-t).*cos(2*pi*t);
El símbolo @ identifica la expresión como una función anónima, a la que se asigna el nombre f. Una vez definida, f(t) puede evaluarse simplemente pasando los valores de entrada de interés como se muestra acontinuacion
t = 0; f(t)
ans = 1
f(0)
ans = 1
Las entradas vectoriales permiten evaluar varios valores simultáneamente.
t = (-2:2);
f(t)
ans = 1×5

1.11-2 Relational Operators and the Unit Step Function

Sueden representar la función de paso unitario, un operador relacional compara dos elementos. Si la comparación es verdadera, se devuelve un verdadero lógico. Los operadores relacionales indican si una condición es verdadera. Existen seis operadores relacionales
disponibles: <, >, <=, >=, == y ~=.
La función de paso unitario se define fácilmente utilizando el operador relacional >=.
u = @(t) 1.0.*(t>=0);
